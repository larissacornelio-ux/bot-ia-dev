# Bot IA Dev

Bot simples que gera e revisa código automaticamente.

## Funcionalidades
- Geração de código a partir de texto
- Revisão automática de código

## Tecnologias
- Node.js

## Como rodar

1. Instale as dependências:
npm install prompt-sync

2. Execute o projeto:
node bot.js

## Objetivo
Projeto criado para praticar desenvolvimento com IA.
